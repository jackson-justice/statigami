# Bees Were More Efficient

**Same chances. Better execution.**
Salt Lake Bees vs River Cats • April 3, 2026

---

## Question

Do more opportunities (baserunners) actually lead to more runs?

---

## Data

* Inning-level game data tracked manually and from box score
* Metrics used:

  * Baserunners
  * Runs
  * Left on base (LOB)

---

## Method

Calculated:

* **Conversion Rate** = Runs / Baserunners
* **Strand Rate** = LOB / Baserunners

This measures how efficiently each team turned opportunities into runs.

---

## Result

* Bees: **37.5%** (6 runs / 16 baserunners)
* River Cats: **29.4%** (5 runs / 17 baserunners)

Both teams had nearly identical opportunities, but the Bees converted at a significantly higher rate.

---

## Takeaway

More chances don’t guarantee more runs. Execution matters.
