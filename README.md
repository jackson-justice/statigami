# Statigami

Statigami is a collection of data-driven projects focused on uncovering rare, surprising, and insightful patterns in sports and everyday life.

Inspired by *scorigami*, this project explores questions like:

* What outcomes or patterns have never happened before?
* Which stats should predict success but don’t?
* Where does real-world data contradict intuition?

Each project combines data analysis, visualization, and storytelling, often turned into short-form content.

---

## Repository Structure

Each project follows a consistent structure:

```
project-name/
│
├── data/        # Raw and/or cleaned datasets
├── scripts/     # R/Python scripts used for analysis
├── figs/        # Visualizations and outputs
├── README.md    # Project-specific explanation
```
